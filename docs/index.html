What is permon?

Permon is a tool to display live charts in a clear, uncluttered way. It is developed with a focus on only showing you things you care about, not everything you can monitor in your system. Permon is designed to work everywhere. It supports Windows, Linux and Mac OS.
But what if you don't have access to a desktop manager? Or want to check performance of your workstation remotely? Permon has you covered. There are 3 frontends. A terminal frontend, native frontend and browser frontend.
